## 2. Logo

### 2.1 Primary Logo

The XGrowthOS logo consists of:
- **Wordmark:** "XGrowthOS" in Sora Bold
- **Icon:** Stylized "X" with growth arrow motif

### 2.2 Logo Variations

| Variation | Use Case |
|-----------|----------|
| **Full Color** | Primary usage on white/light backgrounds |
| **Reversed** | On dark backgrounds |
| **Monochrome** | Single-color applications |
| **Icon Only** | Small applications, favicons, social avatars |

### 2.3 Clear Space

Maintain clear space around the logo equal to the height of the "X" in XGrowthOS.

### 2.4 Minimum Size

| Format | Minimum Width |
|--------|---------------|
| Digital | 120px |
| Print | 1 inch (25mm) |
| Icon only | 32px digital / 0.25 inch print |

### 2.5 Logo Don'ts

❌ Don't stretch or distort
❌ Don't rotate
❌ Don't change colors arbitrarily
❌ Don't add effects (shadows, gradients)
❌ Don't place on busy backgrounds
❌ Don't recreate in different fonts

## 5. Iconography

### 5.1 Icon Style

- **Style:** Outlined, 1.5-2px stroke
- **Corners:** Slightly rounded (2px radius)
- **Size:** 16px, 20px, 24px, 32px grid
- **Source:** Lucide Icons (primary), custom where needed

### 5.2 AI Engine Icons

| Engine | Icon Concept |
|--------|--------------|
| **Guardian** | Shield |
| **Architect** | Blueprint/Compass |
| **Scientist** | Flask/Microscope |
| **Hunter** | Target/Crosshair |
| **Sentinel** | Eye/Radar |

### 5.3 Icon Colors

- **Default:** Inherit from text color
- **Interactive:** Primary color on hover
- **Status:** Use semantic colors

## 6. Imagery

### 6.1 Photography Style

- **Tone:** Professional, modern, tech-forward
- **Subjects:** Diverse business professionals, modern offices
- **Treatment:** Natural lighting, clean compositions
- **Avoid:** Cheesy stock photos, outdated technology

### 6.2 Illustrations

- **Style:** Minimal, geometric, abstract
- **Colors:** Use brand palette
- **Themes:** Growth, connection, automation, intelligence

### 6.3 Data Visualization

- **Charts:** Clean, minimal, using brand colors
- **Dashboards:** Dark mode preferred for product screenshots
- **Metrics:** Highlight with primary color

## 7. UI Components

### 7.1 Buttons

**Primary Button:**
- Background: Emerald (#059669) / Green (#22C55E)
- Text: White
- Border radius: 6px
- Padding: 12px 24px
- Hover: Darker shade

**Secondary Button:**
- Background: Transparent
- Border: 1px solid primary
- Text: Primary color
- Hover: Light primary background

**Ghost Button:**
- Background: Transparent
- Text: Gray/Neutral
- Hover: Subtle background

### 7.2 Cards

- Background: Surface color
- Border: 1px solid border color (optional)
- Border radius: 8-12px
- Shadow: Subtle (light mode only)
- Padding: 16-24px

### 7.3 Forms

- Input height: 40-44px
- Border radius: 6px
- Border: 1px solid border color
- Focus: Primary color border + glow
- Error: Error color border

## 9. Motion

### 9.1 Animation Principles

- **Purpose:** Guide attention, provide feedback
- **Duration:** 150-300ms for UI, 300-500ms for transitions
- **Easing:** ease-out for entrances, ease-in for exits
- **Subtlety:** Enhance, don't distract

### 9.2 Common Animations

| Element | Animation |
|---------|-----------|
| Button hover | Scale 1.02, color transition |
| Card hover | Slight lift (translateY -2px) |
| Modal entrance | Fade in + scale from 0.95 |
| Page transition | Fade (300ms) |
| Loading | Pulse or spinner |
