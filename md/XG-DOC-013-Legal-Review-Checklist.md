## Purpose

This checklist ensures all client engagements and campaigns meet legal and regulatory requirements. Complete before launching any new client or campaign.

---

## Section 1: Client Onboarding Legal Review

### 1.1 Agreement Verification

**Pilot Service Agreement (XG-DOC-003):**
- [ ] Agreement fully executed by both parties
- [ ] Effective date clearly specified
- [ ] Client company name and address accurate
- [ ] Authorized signatory confirmed
- [ ] All exhibits attached (Pricing, SLA)
- [ ] Countersigned copy returned to client

**Mutual NDA (XG-DOC-012):**
- [ ] NDA executed before sharing sensitive information
- [ ] Term and survival periods noted
- [ ] Filed in client folder

**Data Processing Agreement (XG-DOC-022):**
- [ ] DPA executed (required for EU/UK/CA clients)
- [ ] Sub-processors list provided
- [ ] Data retention terms specified
- [ ] Client signed as Controller

### 1.2 Client Entity Verification

- [ ] Business entity type confirmed (Corp, LLC, etc.)
- [ ] Business registration verified
- [ ] Authorized representative confirmed
- [ ] Business address verified
- [ ] Website matches business claims

---

## Section 2: Email Compliance - CAN-SPAM

### 2.1 Sender Identification

- [ ] "From" name clearly identifies sender
- [ ] "From" email domain matches sender identity
- [ ] Reply-to address is monitored
- [ ] No misleading header information

### 2.2 Subject Line Compliance

- [ ] Subject line accurately reflects content
- [ ] No deceptive subject lines
- [ ] No false urgency or misleading claims

### 2.3 Content Requirements

- [ ] Physical postal address included
- [ ] Clear identification as advertisement (if applicable)
- [ ] No false or misleading content

### 2.4 Opt-Out Compliance

- [ ] Unsubscribe mechanism present
- [ ] Unsubscribe clearly visible
- [ ] Opt-out honored within 10 business days
- [ ] No fee or requirements beyond email/opt-out reason
- [ ] Opt-out list maintained and updated

---

## Section 3: GDPR Compliance (EU/UK)

### 3.1 Lawful Basis Assessment

**For B2B Outreach - Legitimate Interest:**
- [ ] Legitimate interest assessment documented
- [ ] Business context for outreach established
- [ ] Minimal data processed (name, email, company, title)
- [ ] No sensitive/special category data
- [ ] Balance test completed (LIA)

### 3.2 Data Subject Rights

- [ ] Right to access: process in place
- [ ] Right to rectification: process in place
- [ ] Right to erasure: process in place (72-hour SLA)
- [ ] Right to object: honored immediately
- [ ] Right to portability: process in place

### 3.3 DPA Requirements

- [ ] Data Processing Agreement executed
- [ ] Processor obligations defined
- [ ] Security measures documented
- [ ] Sub-processor list provided
- [ ] Breach notification process (72 hours)

### 3.4 Technical Measures

- [ ] Data encryption in transit (TLS)
- [ ] Data encryption at rest
- [ ] Access controls in place
- [ ] Audit logging enabled
- [ ] Data minimization practiced

## Section 4: CCPA/CPRA Compliance (California)

### 4.1 Classification

- [ ] XGrowthOS classified as "Service Provider"
- [ ] No "sale" of personal information
- [ ] Business purpose documented

### 4.2 Consumer Rights

- [ ] Right to know: process in place
- [ ] Right to delete: process in place
- [ ] Right to opt-out: process in place
- [ ] Right to non-discrimination: confirmed
- [ ] Right to correct: process in place

### 4.3 Contractual Requirements

- [ ] Service provider terms in agreement
- [ ] Prohibited uses specified
- [ ] Audit rights preserved
- [ ] Certification obligations included

## Section 5: Canada CASL Compliance

### 5.1 Consent Verification

**For B2B Messages:**
- [ ] Implied consent basis documented
- [ ] Business relationship or inquiry exists
- [ ] 6-month/2-year implied consent window applies
- [ ] Or express consent obtained

### 5.2 Message Requirements

- [ ] Sender identification present
- [ ] Contact information included
- [ ] Unsubscribe mechanism functional
- [ ] Opt-out honored within 10 days

## Section 6: Do-Not-Contact (DNC) Compliance

### 6.1 Pre-Send Verification

- [ ] Guardian DNC check completed
- [ ] Scrubbed against client customer list
- [ ] Scrubbed against client competitor list
- [ ] Checked against previous opt-outs
- [ ] Checked against complaint history

### 6.2 DNC List Management

- [ ] Master DNC list maintained
- [ ] Client-specific DNC list active
- [ ] Opt-out additions processed same-day
- [ ] Complaint additions processed immediately
- [ ] DNC list audit monthly

---

## Section 7: Domain & Sender Compliance

### 7.1 Domain Authentication

- [ ] SPF record configured and passing
- [ ] DKIM record configured and passing
- [ ] DMARC policy published (p=quarantine minimum)
- [ ] DNS propagation confirmed
- [ ] Authentication test passed

### 7.2 Sender Reputation

- [ ] Domain not on blacklists
- [ ] IP not on blacklists
- [ ] Domain age >30 days
- [ ] Warmup protocol completed
- [ ] Reputation score >70

### 7.3 Gmail/Yahoo 2024-2025 Requirements

- [ ] One-click unsubscribe header
- [ ] List-Unsubscribe-Post header
- [ ] Spam rate <0.1% (monitored)
- [ ] Authentication passing
- [ ] Valid PTR records

## Section 8: Content Compliance Review

### 8.1 Prohibited Content

- [ ] No false claims or misrepresentations
- [ ] No guaranteed results promises
- [ ] No competitor defamation
- [ ] No unauthorized testimonials
- [ ] No copyrighted content (without license)

### 8.2 Industry-Specific

**Financial Services:**
- [ ] No investment advice
- [ ] Required disclaimers present

**Healthcare:**
- [ ] No medical claims
- [ ] HIPAA considerations addressed

**Legal:**
- [ ] Attorney advertising rules followed
- [ ] State-specific requirements met

### 8.3 Client Approval

- [ ] Copy approved by client
- [ ] Approval documented (email/signature)
- [ ] Approval date recorded
- [ ] Version approved matches deployed
