### Screen Recording Settings
**OBS Studio Configuration:**
- Resolution: 1920x1080
- Frame Rate: 30 FPS
- Format: MP4
- Quality: High

**Recording checklist:**
- Clear browser cache
- Close unnecessary tabs
- Use demo data (not real client info)
- Record in full-screen mode

### Voice Recording Tips
- **Environment**: Quiet room, no echo
- **Equipment**: Phone or USB microphone  
- **Settings**: 44.1kHz, 16-bit minimum
- **Delivery**: Confident, professional tone

### Video Editing Workflow

**DaVinci Resolve Project Setup:**
1. Create new project: "Wryko Demo"
2. Timeline resolution: 1920x1080, 30fps
3. Import all assets to media pool
4. Color correct screen recordings for consistency

**Assembly Order:**
1. Lay down voiceover track first
2. Add screen recordings to match narration
3. Insert graphics slides between sections
4. Add text overlays for key statistics
5. Apply smooth transitions (0.5s cross-dissolve)
